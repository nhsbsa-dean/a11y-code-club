Accessibility Code Club

In this prototype you will find accessibility issues which are common across our services, can you find them all and implement a fix.

Prerequisite

  • Chrome Browser
  • WAVE tool, install the wave tool (Opens in a new tab)
  • AXE tool, install the axe tool (Opens in a new tab)
  • Lighthouse, installed with Chrome browser
  • Voice Over (Screenreader) for macbook users
  • NVDA (Screenreader for windows users

What to do

For each screen you come to, there may be accessibility errors, alerts or both. Use the tools listed in the prerequisite to find these.

Sometimes an accessibility issue is not raised by a tool and requires manual checking, can you spot these too.

Errors must be fixed, alerts should be fixed but can be passed with good reason.
